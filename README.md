# 基于Socket局域网通信软件开发
## 开发环境
	Qt5.5.0、arm-linux交叉工具链
## 项目描述
	    该项目是我在Github发布的第一个项目，接下来会不定期更新。
	    主要解决的问题是跨平台的局域网内文件共享和即时通信，具有
	良好的人机交互界面，其设计宗旨是强实用性，并且能有优秀的人机
	交互界面。
	    软件（取名Singko）实现局域网内各主机间进行聊天通信以及文 
	件互传，支持Windows、 Linux平台，支持ARM平台(移植S5PV210)。
